# Personal Finance Tracker
A Python application that tracks daily expenses using Pandas and visualizes data with Matplotlib and it automatically stores data in a CSV file using **Pandas** and generates visual insights with **Matplotlib**

- **User Validation:** Prevents crashes by checking input types (`.isalpha()` for categories, `.isdigit()` for amounts).
- **Data Persistence:** Automatically creates and updates `xercler.csv` without overwriting historical data.
- **Data Analysis:** Groups expenses by category and calculates totals.
- **Data Visualization:** Generates a clean, colorful **Pie Chart** directly in the console.

## Used
- **Python 3**
- **Pandas** (Data manipulation)
- **Matplotlib** (Data visualization)
- **OS & Time** (Built-in Python modules)
